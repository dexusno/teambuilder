# Creative / Content Pattern - Collaboration

## Style: Creative Process with Review Gates

**Ideation Sessions:**
- Strategist facilitates brainstorming
- All agents contribute ideas
- Audience Analyst provides audience insights
- Collaborative and open-ended

**Creation Phase:**
- Writer works independently
- Creates draft without committee interference
- Trust writer's craft during creation

**Review Gates:**
- Gate 1: Editorial Review (structure, clarity, quality)
- Gate 2: Brand Review (voice, tone, alignment)
- Gate 3: Optimization Review (SEO, formatting)
- Each gate provides specific feedback

**Revision Cycles:**
- Writer incorporates feedback
- May go through 2-3 revision cycles
- Each cycle improves specific aspects

**Approval Flow:**
- All review gates must approve
- Editor has veto on quality
- Brand Guardian has veto on voice
- Strategist makes final publication call

**Creative Freedom:**
- Writer has creative control during drafting
- Reviews provide feedback, not dictation
- Collaborative refinement, not top-down editing
- Ego-free feedback environment

**Structured creative process - ideation freedom, creation focus, multi-layered review, collaborative refinement.**
